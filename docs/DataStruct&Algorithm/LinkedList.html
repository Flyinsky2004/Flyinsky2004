链表 ​

介绍

链表数据结构以头结点，表长构成。结点数据结构由结点数据、前继结点指针、后继结点指针构成。因其内存分布并不连续，故使用指针来互相联系。后文会和数组比较说明。

链表与数组是一对好兄弟，两者的优劣截然不同。数组在内存空间上分布连续，因而在访问元素时可以直接通过下标访问，时间复杂度做到O(1)；而链表在内存空间上分布并不连续，其链表各元素，也称之为结点，之间通过指针来联系。因每个节点中包含的结点个数不同，链表也有了区分（仅有后继结点的链表称之为单链表，具有前继结点和后继节点的链表称之为双向链表）。因这个特性，故在访问链表第n个元素需要从头遍历找到第n个元素，时间复杂度为O(n)。

1.链表及结点结构 ​

介绍中已提及关于链表的组成和数据结构大概，以下是一个C++实现的简单的双向链表：

c++
#include<iostream>
using namespace std;
struct Node{
    int data;
    Node* former;
    Node* next;
    Node(int x) : data(x), next(nullptr),former(nullptr){
        
    }
}
struct LinkedList{
    Node* head;
    int length;
}

以上即是一个简单的链表结构的定义。从初始化赋值函数中易得我们传入了一个整型变量作为结点元素赋值，并为前继结点和后继结点的指针赋值为空指针。结点结构中是否包含former决定链表是否为一张双向链表。链表结构中是否包含length的整型变量记录长度决定着链表是否为一张循环链表。

2.链表的遍历 ​

(1)单链表、双链表的顺序遍历 ​

若想顺序完整遍历一张链表，可以使用while循环，以当前遍历结点是否为空作为退出条件。C++实现代码如下：

c++
void printLinkedList(LinkedList*& list){
    if(list==nullptr){
        cout << "链表为空！" ;
        return;
    }
   	//判断链表是否为空
    Node* p = list->head; 
    //定义一个指针指向链表头节点用于遍历，不能直接使用链表或结点结构中的指针遍历
    while(p!=nullptr){ 
        //以当前结点是否为空作为退出条件，若能进入循环则当前结点必不可能为空
        cout << p->data << ' ' ; //输出结点元素数据
        p = p->next;			//数据输出完成，将遍历指针指向下一个结点
    }
    return;
}

(2)循环链表的顺序遍历 ​

循环链表

如图所示循环链表与普通链表的区别即普通链表的尾结点指向空，而循环链表的尾结点指向头结点，达成循环结构。

若我们使用传统的遍历方式遍历循环链表则会出现死循环的情况，因为传统遍历方式退出条件为判断遍历的结点是否为空，但循环链表遍历到尾部又指回头节点。因此我们可以使用循环链表结构中的长度属性以计数的方式判断是否遍历完整张表。

以下是C++实现代码：

c++
void printCirculate(LinkedList*& list){
    if(list == nullptr){
        cout << "链表为空!";
        return;
    }
    Node* p = list->head;//同样定义一个用于遍历的指针
    int i = 0;//定义记录遍历长度的整型变量
    while(i < list->length){
        //类似与数组遍历，结束条件为i是否大于等于表长
        cout << p->data << ' ' ;
        //输出数据元素
        i++;
        p = p->next ;
        //自增遍历长度 使指针下指
    }
    return;
}

3.链表增删改 ​

(1).增加结点 ​

链表插入的大致步骤即为初始化一个新的结点，遍历到链表尾部将尾结点的后继节点指针指向新创建的结点，若链表为双链表则还需将新尾部结点的前继结点指针指向原尾节点。

c++
void linkList_insert(Node*& h,int x) {
	if (h == nullptr) {
		Node* p = new Node(x);
		h = p;
	}
    //判断头结点是否为空 若为空则直接将新节点作为头结点
	else {
		Node* p = h;//新建遍历结点
		while (p->next != nullptr)
		{
			p = p->next;
		}//遍历到尾结点
		Node* n = new Node(x);//新建结点
		n->former = p;//完成指向
		p->next = n;
	}
}

(2).删除结点 ​

删除节点步骤如下:

  1. 先判断表是否为空
  2. 定义遍历指针遍历整张表，定义一个操作指针用于符合删除条件时便于操作
  3. 若满足删除条件，则使r指针指向的节点(待删除节点)的前继结点的后继结点指向待删除节点的后继结点，使待删除节点的后继结点的前继结点指向待删除节点的前继结点。这样便使在之后的顺序遍历时会跳过待删除节点，最后使用c++操作符delete删除r结点，释放内存。

以下是一个C++实现的删除所有结点值为x的结点:

c++
void deleteByXY(Node*& h, int x, int y) {
	if (h == nullptr) {
		cout << "链表为空，非法操作！" << endl; 
		return;
	}//判断表是否为空
	Node* p = h;//定义遍历指针用于遍历
	while (p != nullptr) {
        Node* r = p;
        p = p->next;
		if (r->value > x && r->value < y) {
			r->former->next = r->next;
			r->next->former = r->former;
            delete r;
		}
	}
}

(3).修改结点 ​

i.若只是简单的修改结点的数据则很简单，只需要遍历到该元素直接使用赋值语句修改即可，这里不做演示。

ii.在不改变原有内存空间的情况下，交换一个结点和其前一个结点的位置。

步骤：

以下是C++实现代码：

c++
void changePosition(CirculateNode*& c , int x) {
	if (c->head == nullptr || c == nullptr) {
		cout << "双向循环链表为空，非法操作！";
		return;
	}
	if (x - 1 > c->length) {
		cout << "非法位置！";
		return;
	}
	Node* p = c->head,* tail = c->head;
	int i = 0;
	//while (tail->next != nullptr) tail = tail->next;
	for (int i = 0; i < x; i++) p = p->next;
	p->former->former->next = p;
	p->former->next = p->next;
	p->next->former = p->former;
	p->former->former = p;
	p->next = p->former;
	p->former = p->former->former;
}
